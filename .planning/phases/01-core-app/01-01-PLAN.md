          create_id()
      }

          tauri::Builder::default()
              .plugin(tauri_plugin_clipboard_manager::init())

  **Implementation (GREEN phase):**
  Replace scaffold's `src/App.tsx` with a single `App` component:
  - State: `cuid: string | null` (initially null), `copied: boolean` (initially false)
  - `handleGenerate` async function: calls `invoke<string>('generate_cuid')`, then `writeText(id)`, then `setCuid(id)`, `setCopied(true)`, `setTimeout(() => setCopied(false), 1500)`
